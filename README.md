# Gridlington game -- catch the square!

## About
A little browser game in HTML, CSS, and Javascript. Click the colored square! It will turn pick if you catch it.

A game version of the original [Gridlington](https://github.com/jwjacobson/gridlington).

## Installation/Use
You can play at the [Github pages page](https://jwjacobson.github.io/gridlington-game/) of this repo.

If you'd like to mess around with the code, just clone the repo, things should be simple enough.
